Suede developer docs explain the public interfaces, rights concepts, and agent-commerce discovery surfaces in the Suede ecosystem. The canonical documentation is maintained in the Suede-AI/suede-docs repository on GitHub.

What is Suede Labs AI?

Suede Labs AI builds creator-ownership infrastructure for AI-native media. The public docs cover programmable IP, provenance, rights metadata, licensing workflows, and x402 and ACP discovery surfaces. AI music is one implementation area, not the whole product.

What are Suede's core pillars?

  • Programmable IP: interfaces for recording provenance, rights metadata, and machine-readable usage terms.
  • Creator ownership: documentation for connecting creative work to creator-supplied identity and rights context.
  • Provenance: source and metadata records that help people and software inspect where a media asset came from.
  • Licensing: documented workflows for publishing terms and reviewing license activity.
  • Agent commerce: public x402 and ACP discovery files that describe available services and current prices.
