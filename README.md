# Suede Docs Site

Redirect page for `docs.suedeai.ai` — Suede Labs AI developer documentation lives on GitHub at [Suede-AI/suede-docs](https://github.com/Suede-AI/suede-docs) (mirrored publicly at [JasonColapietro/suede-docs](https://github.com/JasonColapietro/suede-docs)).

This is a single static `index.html` that redirects visitors to the canonical docs repo. No build step.

## Deploy

Deployed on Vercel from `index.html` + `vercel.json`.
